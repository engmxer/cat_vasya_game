# RPG игра
## О проекте
Это RPG игра на Python в которой есть: система классов, система заклинаний(спеллов), а также личные характеристики каждого из класса, от который может зависеть исход игры.

## Первый модуль. Создание абстрактной модели и класса Unit

Сам класс содержит всех юнитов игры, а также их:
- 6 характеристик(сила, ловкость, телосложение, мудрость, интеллект и харизму)
- Абстрактные методы которые высчитывают максимальное здоровье персонажа ('calculate_max_health()'),их урон('calculate_damage()') и их защиту('calculate_defense()')
- Класс содержит список заклинаний ('spells') и текущую ману героя ('mana')
- Содержит метод 'add_spell' для добавлению персонажу заклинаний
- Метод 'cast_spell()' для использования заклинаний с проверкой маны

## Второй Модуль. Класс персонажа (Class Character)

В данном модуле создаются сами классы игровых персонажей:
- Класс **Воин** или же **warrior**: урон этого класса конкретно зависит от силы и телосложения. Воин носит тяжелую броню, ему ловкость не особо важна
- Класс **Маг** или же **mage**: урон зависит от интеллекта и мудрости. Маги ставят магические барьеры, а не уворачиваются
- Класс **Охотник** или же **hunter**: урон зависит от ловкости и силы. Суть охотника в том, что он уклоняется от атак, а не принимает их на грудь.
Формулы высчитывания характеристик для каждого из класса
- **Воин**
    - **Урон**: сила * 2.2 + телосложение // 3
    - **Защита**: телосложение * 1.8 + сила // 4
- **Маг**
    - **Урон**: интеллект * 2.5 + мудрость // 2
    - **Защита**: мудрость * 1.3 + интеллект // 6
- **Охотник**
    - **Урон**: ловкость * 1.9 + сила // 3
    - **Защита**: ловкость 1.6 + телосложение //5

## Третий Модуль. Система заклинаний

В этом модуле реализованна система магических заклинаний:
- Абстрактный класс 'Spell' с атрибутами: название, урон, стоимость маны
- Конкретные заклинания
    - **Fireball** или **Огненный шар**: наносит 35 урона и тратит 15 маны
    - **IceLance** или **Ледяное копьё**: наносит 25 урона и тратит 10 маны
    - **LighringBall** или **Шаровая молния**: наносит 40 урона и тратит 20 маны
